# Repo-Local Codex Skills

This directory vendors skills from `mattpocock/skills` for use while building and maintaining this project.

Source: https://github.com/mattpocock/skills
Snapshot: 8868f54212dfcf450b665d2e2a5bf521ada64c3e
License: MIT, included in `mattpocock-skills-LICENSE`.

Most useful build workflows for this app:

- `tdd`: implement or fix behavior with a red-green-refactor loop.
- `qa`: run a structured quality pass before handoff.
- `triage-issue`: investigate a bug and produce a fix plan.
- `improve-codebase-architecture`: find architecture and testability improvements.
- `design-an-interface`: explore UI/API alternatives before implementation.
- `domain-model`: clarify business concepts and data boundaries.
- `request-refactor-plan`: plan a larger refactor in small commits.
- `setup-pre-commit`: add pre-commit checks when the team wants local guardrails.

Use the exact skill name in prompts, for example: `Use tdd to implement candidate search validation`.
